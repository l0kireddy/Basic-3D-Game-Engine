## [0.1.0] - 2025-11-13

### Added
- Initial release of GD3D Editor
- Visual 3D scene editor with drag-and-drop interface
- Three.js integration for 3D graphics rendering
- GLTF/GLB model import support
- Transform controls (translate, rotate, scale)
- Hierarchy panel for scene organization
- Inspector panel for object properties
- Asset browser for resource management
- Physics engine integration using cannon-es
- Collision detection and rigid body dynamics
- Physics debug visualization
- Advanced character controller system
  - Third-person camera
  - WASD movement controls
  - Jump mechanics
  - Animation blending support
- Event sheet system for visual scripting
- Condition-action programming model
- Prefab system for reusable game objects
- Audio system with sound management
- Audio controls and playback
- Post-processing effects
- Supabase integration for backend services
  - User authentication
  - Cloud storage for projects
  - Asset storage and management
- Export system for standalone HTML5 games
- Play mode for in-editor testing
- Project save/load functionality
- Example project: Simple platformer
- Comprehensive documentation

### Technical
- React 18 with Vite build system
- Tailwind CSS for styling
- Zustand for state management
- react-dnd for drag-and-drop functionality
- Lucide React for UI icons
- ESLint configuration
- PostCSS and Autoprefixer

### Documentation
- README with installation instructions
- Supabase setup guide
- Contributing guidelines
- MIT License
- Environment configuration template
